Futari

Política de Privacidade

Última atualização: 11 de maio de 2026

O Futari é um aplicativo privado para duas pessoas organizarem registros compartilhados, como datas, listas, memórias, recados, trackers, fotos, cidades visitadas e áudios de sonhos. Esta política explica quais dados o app trata, por que eles são usados e quais controles estão disponíveis para cada usuário.

Para solicitações de privacidade, suporte ou exclusão de dados, entre em contato pelo e-mail pedroschuetze@hotmail.com.

Quem controla os dados

O Futari é mantido por Pedro Andrade. O canal de contato acima deve ser usado para solicitações relacionadas a privacidade, suporte e exclusão de conta.

Dados que coletamos

O Futari coleta e armazena somente os dados necessários para o funcionamento do app:

  • dados de conta: e-mail, identificador de usuário, nome exibido, foto de perfil opcional e data de nascimento opcional;
  • dados do casal: nome do espaço compartilhado, código de convite e vínculo;
  • conteúdo criado no app: listas, itens, datas, eventos, recados, memórias, fotos, trackers, notas, avaliações, cidades visitadas, livros, vinhos, apostas e sonhos;
  • mídias enviadas pelo usuário: fotos de perfil, fotos de memórias, fotos de trackers e áudios gravados na seção de sonhos;
  • dados técnicos básicos: identificadores internos, datas de criação/atualização e informações necessárias para sincronizar conteúdo entre os dois membros.

O app não coleta dados de localização em tempo real. A seção de cidades usa apenas as cidades que o usuário registra manualmente ou seleciona na busca do app.

Como usamos os dados

  • autenticar o usuário e manter a sessão ativa;
  • criar e manter o espaço privado da dupla;
  • sincronizar conteúdo entre os dois membros;
  • exibir fotos, áudios, listas, datas e trackers dentro do app;
  • permitir edição de perfil, saída do espaço compartilhado, logout e exclusão de conta;
  • manter segurança, integridade e funcionamento básico do serviço.

Não vendemos dados pessoais. Não usamos dados do Futari para publicidade comportamental. Não compartilhamos conteúdo privado com outros usuários.

Compartilhamento com provedores

O Futari usa serviços de infraestrutura para operar:

  • Supabase: autenticação, banco de dados, armazenamento de arquivos e sincronização;
  • Expo/EAS: build, atualizações do app e distribuição;
  • Apple: distribuição via TestFlight/App Store e serviços relacionados ao sistema operacional.

Esses provedores processam dados somente para viabilizar o funcionamento técnico do app, de acordo com seus próprios termos e políticas.

Fotos, microfone e permissões

O app pode pedir acesso às fotos para permitir que o usuário selecione imagens para perfil, memórias, listas, cidades, vinhos ou livros. O app pode pedir acesso ao microfone para gravar áudios na seção de sonhos.

Essas permissões são opcionais. Se o usuário negar uma permissão, as demais funcionalidades continuam disponíveis, mas os recursos que dependem daquela permissão podem ficar limitados.

Retenção e exclusão

Os dados permanecem armazenados enquanto a conta estiver ativa ou enquanto forem necessários para manter o espaço compartilhado.

Dentro do app, o usuário pode sair da conta, sair do espaço compartilhado e excluir a própria conta pessoal.

Ao excluir a própria conta, o Futari remove dados pessoais ligados ao usuário, como perfil, foto, data de nascimento, progresso individual e avaliações individuais. Como o app é compartilhado por duas pessoas, conteúdos do espaço compartilhado podem permanecer preservados para o outro membro, salvo quando a exclusão específica daquele conteúdo estiver disponível no próprio app.

Segurança

O Futari usa autenticação por Supabase Auth e políticas de acesso no banco de dados para limitar o conteúdo ao espaço compartilhado correspondente. Arquivos privados são armazenados em buckets com regras de acesso relacionadas ao usuário e à dupla.

Apesar dessas medidas, nenhum sistema é completamente imune a riscos. Se você suspeitar de acesso indevido, entre em contato pelo canal de suporte acima.

Crianças

O Futari não é destinado a crianças. O app deve ser usado por pessoas com idade suficiente para consentir com a criação de conta e o armazenamento dos próprios dados de acordo com as leis aplicáveis.

Mudanças nesta política

Esta política pode ser atualizada conforme o Futari evoluir. Quando houver mudanças relevantes, a data de atualização será alterada e a versão mais recente ficará disponível nesta página.
